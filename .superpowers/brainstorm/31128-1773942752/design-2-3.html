Agent Loop

Diagram description

A loop with three steps: Gather (read context, scan tools, assess state), Act (plan steps, call tools, execute), and Observe (check results, decide next move). An arrow labeled "repeat" returns from Observe back to Gather.

How it's generated: pages.clj scans /assets/*.svg at build time. For each SVG, it finds which post(s) reference it (back-link), derives the title from the filename, and generates a /diagrams/<slug>/ page. The alt text from the referencing post's <img> tag becomes the transcript text.
