SVG diagrams — what should "open standalone" mean?

Currently: diagrams are inlined SVGs with no click affordance. On mobile they render at ~354px wide (46% of design size). Desktop looks great at 720px.

Mobile now (354px)
no click affordance
A

Lightbox — overlay on the same page

Tap diagram → full-screen dark overlay with the SVG scaled to viewport width. Tap anywhere or press Escape to dismiss. No navigation, instant, works on mobile and desktop. Needs ~30 lines of JS + CSS.

B

Dedicated page per diagram

/diagrams/agent-loop/ etc. Built at compile time, full URL, shareable, opens in same or new tab. SVG fills the page. Pure HTML — no JS needed. Requires changes to the build pipeline.

C

Link to raw SVG file

Click wraps to /assets/agent-loop.svg. Browser opens it natively — pinch-to-zoom on mobile, ctrl+scroll on desktop. Zero build changes, zero JS. Rougher experience but simplest.
